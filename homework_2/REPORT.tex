\section{Problem Statement}
Given a noisy full-size image and multiple ROI query images, detect and localize each ROI in the full image using preprocessing, local feature extraction, and feature matching.

\section{Method}
\subsection{Preprocessing}
All images were converted to grayscale and denoised with Gaussian blur.

\subsection{Feature Extraction}
ORB keypoints and descriptors were computed for the full image and each ROI.

\subsection{Matching and Localization}
Descriptor matching was performed with BFMatcher using Hamming distance and Lowe ratio filtering. ROI locations were estimated from matched full-image keypoint coordinates and visualized with rectangles.
